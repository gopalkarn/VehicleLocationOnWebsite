This Project allows to view the location of a vehicle from the database on a map embedded inside the website.
This Project allows to update the Name, Password and Vehicle No. of each customer directly into the database through registration page.
The Location in the database is updated through the gsm module of the "Vehicle Security System" **This code is not included in this project**
This is a simple interface and can be further modified to make it more attractive and eye pleasing.

Software used:
1. Sublime Text
2. XAMPP
